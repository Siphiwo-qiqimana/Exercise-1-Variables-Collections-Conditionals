This test is a structured Python exercise focused on variables, collections, and conditionals, designed for a Data Analytics with Python course using a Databricks environment. It provides a small student dataset (eight records with fields like name, age, subject, score, and passed status) and guides students through three sections. The first section covers creating variables for a single student, identifying data types, performing arithmetic, and using type() and print(). The second section introduces collections: creating lists (student names, scores), a dictionary for one student, and a list of dictionaries for multiple students, then accessing specific values using indexing and keys. The third section involves logical operators and conditional logic: evaluating comparison expressions, combining conditions with and/or/not, writing if/elif/else statements to assign grade categories, and a challenge using combined conditions. A reflection section asks conceptual questions about the differences between lists and dictionaries, explaining logical conditions in plain English, and choosing appropriate collection types for quick name-based lookup. Students are required to write code in Databricks, record outputs in this document, save as PDF, upload both PDF and notebook to GitHub, and submit the link.
